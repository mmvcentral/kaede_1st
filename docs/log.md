## AI Release: 答えろ守矢ァ！！ (Answer Moriya!!) — り〜どみ〜

**Creator:** HAL  
**Version:** 1.04 (First release)

### Notes

- Same-character AI is designed to be fun. Please enjoy.
- Compatibility with same character (Kaede-1st): button layout, key command display, stun/hit affected by opponent, stun or hit cancel, projectile absorption, invincible characters disabled.
- 12P uses A display by default.
